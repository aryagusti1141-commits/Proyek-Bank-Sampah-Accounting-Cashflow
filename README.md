Deskripsi: 
Aplikasi desktop untuk pengelolaan aliran kas dan neraca otomatis Proyek Bank Sampah dibangun dengan memanfaatkan antarmuka JavaFX yang sangat kontemporer. Aplikasi ini juga memanfaatkan database Supabase yang berbasis PostgreSQL. Dalam proses pembuatan, aplikasi ini mengadopsi prinsip Object Oriented Programming atau OOP, dan menggunakan tipe data desimal yang sangat presisi. Ini dilakukan untuk mempertahankan ketepatan penghitungan uang dalam Proyek Bank Sampah.
