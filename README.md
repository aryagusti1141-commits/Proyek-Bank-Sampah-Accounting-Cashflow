# 🍃 Bank Sampah Accounting & Cashflow

Aplikasi desktop cerdas untuk pengelolaan aliran kas (*cash flow*) dan pencatatan neraca otomatis, dirancang khusus untuk operasional Proyek Bank Sampah. 

Aplikasi ini dibangun menggunakan arsitektur **Object-Oriented Programming (OOP)** yang solid dan memanfaatkan antarmuka **JavaFX** kontemporer untuk pengalaman pengguna yang intuitif. Demi mempertahankan tingkat presisi dan akurasi mutlak dalam penghitungan finansial, sistem ini secara ketat menggunakan tipe data desimal presisi tinggi dan terintegrasi dengan basis data *cloud* PostgreSQL dari **Supabase**.

## 🛠️ Teknologi yang Digunakan

*   **Bahasa Pemrograman:** Java 
*   **GUI Framework:** JavaFX
*   **Database:** PostgreSQL (via Supabase Connection Pooler)
*   **Dependency Management:** Maven
*   **Environment Manager:** `dotenv-java` (io.github.cdimascio)
